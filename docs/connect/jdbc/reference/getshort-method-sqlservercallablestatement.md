# <a name="getshort-method-sqlservercallablestatement"></a>getShort-Methode (SQLServerCallableStatement)

  Ruft den Wert des angegebenen Parameters als Wert vom Typ **short** in der Programmiersprache Java ab.

|----------|-----------------|
